# 專案作品開發規劃與時程管理系統 v1.4

這是一個專為畢業展覽作品開發規劃設計的單頁式 Web 應用工具。透過網頁介面，使用者可以輕鬆管理作品清單、設定製作參數、預估時程，並產出專業的規劃報表。

## 🚀 主要功能

-   **作品管理系統**：支援新增、編輯、刪除多項作品，並可自定義分類。
-   **智慧參數計算**：內建緩衝比例 (Buffer Ratio) 與製作週期計算，自動估算各階段所需時間。
-   **進度追蹤與備註**：整合備註區域，方便記錄每個作品的細節或修改方向。
-   **資料持久化**：
    -   **本地存儲 (Local Storage)**：自動保存當前編輯狀態，重新整理不失。
    -   **JSON 導出/導入**：支援將規劃數據存為 `.json` 檔案，方便進行備份或與他人協作。
-   **列印最佳化**：內建專用列印樣式 (Print Media Query)，可直接產出乾淨、美觀的紙本報表。

## 🛠 使用技術

-   **HTML5 / CSS3**：採用現代化的藍色調介面設計 (Responsive Design)。
-   **Pure JavaScript**：無須安裝任何依賴套件，開啟 HTML 即可使用。
-   **Base64 緩存機制**：優化小型資產處理。

## 📖 使用說明

1.  **開啟工具**：直接使用瀏覽器開啟 `Objectv1.4.html`。
2.  **設定參數**：在頂部或側邊欄設定基礎緩衝時間與主題分類。
3.  **編輯作品**：點擊「新增作品」填寫名稱、難度或預估工時。
4.  **保存與匯出**：
    -   點擊「保存」可將檔案下載為 JSON。
    -   點擊「讀取」可載入先前的 JSON 規劃檔。
5.  **列印報表**：按下 `Ctrl + P` (或 `Cmd + P`)，系統會自動隱藏按鈕與側邊欄，僅顯示整潔的表格供列印。
